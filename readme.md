# Project Sekai Website with React, Tailwind, and Node.js + express

## Quick Start (Front end)

```bash
git clone https://github.com/DarrenCasper/Project-Sekai-Website
cd front-end
npm install
npm run dev
```
Your website will be available at [http://localhost:5173](http://localhost:5173)


## Quick Start (back end)

- Make sure to download node.js before trying to run this proram

```bash
git clone https://github.com/DarrenCasper/Project-Sekai-Website
cd backend
npm install
cd src
node server.js 
```

Your backend localhost will be available at [http://localhost:5000](http://localhost:5000)
